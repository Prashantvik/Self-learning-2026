#### Unified Modeling Language (UML) Diagrams

##### Class diagram -
- Class diagram is a visual representation of the classes and their
relationships in a system. It is a type of static structure diagram that shows
the classes, their attributes, methods, and the relationships between them.
- Class diagrams are used in object-oriented design to model the structure of a
system and to communicate the design to stakeholders. They can be used to
identify the classes and their relationships, to design the class hierarchy,
and to identify the responsibilities of each class.
